Hardware

Hardware refers to the physical components of a computer, like the CPU, RAM, and keyboard.

Software

Software is the set of instructions that tells the hardware what to do, like Windows or Web Browsers.

Networking

Networking connects computers together to share resources, usually through the Internet.
